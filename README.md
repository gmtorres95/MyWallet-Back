## How to run

### Install the application

```bash
# Clone this repository
$ git clone https://https://github.com/gmtorres95/MyWallet-Back

# Install the dependencies
$ npm i
```

### Configure the .env file

Use the .env.example file

### Run the application

```bash
$ npm run start
```
